# m1-ayx-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容各自独立，不针对任何特定网站或域名，仅作为通用信息页面的集合与展示。

## 项目简介

本项目旨在整理和托管一组静态 HTML 文件，便于用户直接通过浏览器访问。所有页面均为纯前端实现，无需后端服务支持。

## 目录说明

```
m1-ayx-guide/
├── index.html      # 主页入口
├── page1.html      # 示例页面一
├── page2.html      # 示例页面二
└── ...
```

- `index.html`：仓库的主页文件，提供页面导航或简介。
- 其他 `.html` 文件：独立的信息页面，每个文件对应一个独立主题或功能。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖外部数据库或服务器端脚本。
- 页面内容可能包含文本、图片、样式及少量 JavaScript 交互。
- 页面之间无强关联，可单独打开使用。

## 维护说明

- 本仓库会不定期更新，添加新的 HTML 页面或调整现有内容。
- 欢迎提交 Issue 或 Pull Request 来建议新页面或报告问题。
- 请勿在页面中插入违反平台政策的内容。

## 如何使用

1. 克隆或下载本仓库到本地。
2. 直接双击任意 `.html` 文件，即可在浏览器中查看。
3. 也可通过 GitHub Pages 等静态托管服务部署后在线访问。
